- The Contents of a Transaction
	- Transaction Hash:
		- Unique identifier from this transaction
	- Block Number:
		- Block which the transaction was included in
	- From and to wallet addresses
	- Transaction fees:
		- Amount paid to the mine to process the transaction
	- Gas Price:
		- cost per unit of gas specified for the transaction in ether
